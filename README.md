# Glory — Online CV & Portfolio

**Ishimwe Shema Kaneza Glory** (Glory) — Software Developer

Built with **HTML and CSS only** — colors: white, green, blue, and black.

## What's on the page

- **Hero** — photo area, intro, multiple CTAs
- **Stats bar** — impressive numbers at a glance
- **About** — professional summary
- **Experience** — CV timeline (edit with your real jobs/school)
- **Skills** — pills + progress bars
- **Why hire me** — highlight cards
- **Projects** — your work with buttons
- **Contact** — form + quick links
- **Floating "Email me" button** — always visible

## View it

Double-click `index.html` in your browser.

## Save as PDF (your CV)

1. Open the page in Chrome or Edge
2. Click **"Download CV (PDF)"** or press `Ctrl + P`
3. Choose **Save as PDF**
4. Save the file and send it to employers

## Customize

| What to change | Where |
|----------------|--------|
| Your photo | Replace the `G` circle in `index.html` — add `<img src="photo.jpg">` inside `.hero-photo` |
| Colors | Top of `styles.css` in `:root` |
| Experience & education | `#experience` section in `index.html` |
| Projects | `#projects` section |
| GitHub / LinkedIn | Search for `yourusername` |
